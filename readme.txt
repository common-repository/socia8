Manage multiple social media accounts by posting to all at once, you can review post responses like comments, likes, retweets.

== Description ==

Easily publish your WordPress posts or blogs to your social media platforms all at once via socia8, instead of posting it separately for each social media, using this lightweight WordPress plugin.
Once a post has been published, you can track its log/history from your socia8.com account. You can also see your post's popularity by watching comments, likes, retweets, re-pins, +1s, etc in "Posted" page of your socia8.com account.

== Installation ==

* If you don't know how to install a plugin for WordPress, [here's how](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins).

1. Upload the `Socia8` folder to the `/wp-content/plugins/` directory
2. Active the Socia8 plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the `Socia8` menu that appears in your admin menu.

= Requirements =
* WordPress 4.5 or greater, smaller versions maybe also possible, but untested.

== Frequently Asked Questions ==

= Does *socia8* work for posts and pages both? =
No, It works for posts only. Let us know if you have any specific requirement for page and we will be happy to look into it.

= Does *socia8* work for add and edit posts both? =
Yes, It does.

= Steps to use *socia8* ?
1. Login to socia8 from *socia8* menu in your left hand admin panel.
2. You can then connect your Social media accounts, from the same screen. This is required to post the new post/blog information to your social media account.
3. When you are going to add new post, you will find meta-box for socia8, at the bottom of add post screen.
4. Select the social platforms and add schedule time as per your convenience.
